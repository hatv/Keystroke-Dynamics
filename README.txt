Projet d'étude complète sur les possibilités offertes par l'analyse de la dynamique de frappe d'un utilisateur en matière d'identification et d'authentification.
Le projet se décompose en un projet compilé avec GWT et un second ne contenant que l'applet Java.
L'applet Java est utilisé par l'application web pour accéder directement aux données clavier fournies par l'OS.
Auteur : Victor Hatinguais
